# Anton Petrov

I build internal tools, workflow automation, and operations-facing systems.

Most of my work sits between Python, APIs, Google Sheets, browser automation, internal dashboards, and AI-assisted workflows. I am strongest in messy real-world environments where the process already exists, but the system around it is weak, manual, or unreliable.

## What I work on

- internal tools for live business workflows
- workflow automation and recurring task orchestration
- data validation, monitoring, logging, and dashboards
- Google Sheets / Apps Script systems for operations
- browser-driven automation under real constraints
- AI-assisted tools for practical team use, not AI for show

## Public repositories

These are public repos I keep on GitHub. Some are utilities, some are browser tools, and some are standalone experiments.

### Python / automation

- [`Macro-engine`](https://github.com/Rasslabsya4el/Macro-engine)  
  Universal Python macro engine for recording and executing keyboard / mouse workflows with per-window targeting, combos, delays, and loop modes.

- [`Steam-review-parcer`](https://github.com/Rasslabsya4el/Steam-review-parcer)  
  Local research toolkit for crawling Steam products and reviews, then exploring the data through a Flask UI backed by a FastAPI backend.

- [`reddit_vids_scraper`](https://github.com/Rasslabsya4el/reddit_vids_scraper)  
  Python script for pulling videos from selected subreddits and posting them to Telegram on a schedule.

### Browser extensions / lightweight apps

- [`PoE-Wiki-Quick-Search-Extension`](https://github.com/Rasslabsya4el/PoE-Wiki-Quick-Search-Extension)  
  Chrome extension for fast PoE1 / PoE2 wiki search with autocomplete and a UI styled after the official Path of Exile trade site.

- [`PoE-Trade-Multi-Listings-extension`](https://github.com/Rasslabsya4el/PoE-Trade-Multi-Listings-extension)  
  Chrome extension that groups duplicate trade listings by seller, normalizes prices to chaos, and helps jump to the best lot faster.

- [`Youtube-Aim-trainer-extension`](https://github.com/Rasslabsya4el/Youtube-Aim-trainer-extension)  
  Lightweight YouTube overlay game built as a Chrome extension.

- [`7tv-telegram-bot`](https://github.com/Rasslabsya4el/7tv-telegram-bot)  
  Minimal Cloudflare Worker that answers Telegram inline queries with SevenTV emotes without needing a dedicated server.
